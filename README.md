Console Local Chat
Instructions for use
1. Run the Server class in the IDE
2. Specify the port (from 1024 to ~64000, training ports: 4000, 8080, 1500, etc.)
3. Run the Client class (without stopping the server)
4. Specify the local address (127.0.0.1)
5. Specify the port you specified for the server
6. Specify your name (any, but there can only be one unique name in the chat)
7. For testing, you can enable the IDE's feature for running multiple Client classes at once)
8. In a new window, repeat all the steps, and you will see two users in the chat. You can also chat with yourself.
9. Imagine you are Gollum from The Lord of the Rings (who was talking to himself, which is a joke in case anyone didn't get it (Don't make jokes that aren't funny...)

Консольный локальный чат
Инструкция по использованию
1. Запустите в IDE класс Server
2. Пропишите порт (от 1024 до ~64000, учебные порты: 4000, 8080, 1500 и тд)
3. Запустите класс Client (не останавливая сервер)
4. Пропишите локальный адрес (127.0.0.1)
5. Пропишите порт, который указали серверу
6. Пропишите свое имя (любое, но уникальное имя может быть только одно в чате)
7. Для теста, можете включить функцию в IDE по запуску сразу нескольких классов Client)
8. В новом окне повторите все действия, и вы увидите что в чате 2 пользователя. Можно общаться с самим собой
9. Представьте что вы Голлум из Властелина колец (который сам с собой разговаривал, это шутка если кто не понял (Не шутите не смешные шутки...)
