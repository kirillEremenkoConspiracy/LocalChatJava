Локальный чат с графическим интерфейсом
ИНСТРУКЦИЯ

1. Внутри IDE запустить класс Server

2\. Прописать любой порт от 1025 до ~64000 (4040, 4000, 8080...)

3\. Когда сервер будет запущен, запустить класс ClientGuiController

4\. В открывшемся окне, при запросе адреса ввести:

&nbsp;	127.0.0.1 или localhost

5\. Ввести ранее указаный порт
6. Ввести имя



Для запуска чат-бота необходимо запустить класс BotClient



Local chat with a graphical interface

INSTRUCTIONS

Run the Server class within the IDE

2\. Specify any port from 1025 to ~64000 (4040, 4000, 8080, etc.)

3\. Once the server is running, run the ClientGuiController class

4\. In the window that opens, when prompted for an address, enter:

127.0.0.1 or localhost

5\. Enter the previously specified port

6\. Enter a name



To run the chatbot, you need to run the BotClient class
